# Truck-and-Rail-Traffic-Patterns-at-U.S.-Borders

Truck and Rail Traffic Patterns at U.S. Borders

Author: Linh Huynh

Course: DSC680 – Applied Data Science, Bellevue University

1. Project Overview

This project analyzes freight movement across U.S. borders with Canada and Mexico using historical border-crossing data.

The goal is to help logistics companies understand freight patterns, forecast future traffic, and identify which ports require the most resources for planning and investment. 


2. Dataset

Border Crossing Entry Data

Source: Kaggle

The dataset includes:

Port name and state

Border (Canada or Mexico)

Month and year

Type of crossing (truck, rail, passenger)

Number of crossings

The analysis focuses on truck and rail freight traffic, which are most relevant for logistics operations. 


3. Methods

The project includes:

Data cleaning and preprocessing

Exploratory data analysis (EDA)

Freight traffic trend analysis

Forecasting models

Port clustering analysis

Models and techniques used:

Seasonal Naive Forecast

Linear Regression

K-Means Clustering

4. Key Findings

Truck traffic is significantly higher than rail traffic at both borders.

Freight activity follows clear seasonal patterns, with peaks in spring and fall.

Major freight hubs include Laredo, Detroit, and Buffalo–Niagara Falls.

The seasonal naive model performed better than linear regression for forecasting.

Ports can be grouped into small, medium, and major freight hubs using clustering.
